

\subsubsection{\texttt{get\_scope}}

returns the \meta{Scope} reflecting the scope of a scoped declaration reflected by a \meta{Scoped}.

\begin{minted}{cpp}
namespace meta {
\end{minted}
\begin{minted}[xleftmargin=1em,tabsize=4]{cpp}
template <Scoped T>
struct get_scope
{
	typedef /* generated by the compiler */ type;
};
	
template <Scoped T>
using get_scope_t = typename get_scope<T>::type;

\end{minted}
\begin{minted}{cpp}
} // namespace meta
\end{minted}


Note that \texttt{get\_scope<...>::type}
must conform to the \meta{Scope} concept.
